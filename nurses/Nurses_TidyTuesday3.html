Data Filtering


After looking at the top 5 states with the highest Annual Average Salary, I filtered the data-set to select only these 5 states. I call this new dataset States_5 which is the 5 Highest States and a mean Annual Salary Average.

states_5 <- nurses %>%
 filter(State %in% c("California", "Hawaii", "Massachusettes", "Oregon", "Alaska", "District of Columbia")) %>%
 group_by(State) %>%
 summarise(AvgSal = mean(`Annual Salary Avg`)) %>%
 arrange(AvgSal)

Data Visualization 1


The graph below helps us visualise the differences in Average Salaries across these 5 states. We can see that California has the highest, followed by Hawaii, Alaska, Oregon, and DC that lag slightly.

states_5 %>%
 ggplot(aes(x = State,y = AvgSal, fill = State)) + geom_col() +
 labs(title = "States with Highest Average Salaries for Nurses",
 x = "State",
 y = "Average Salary") +
  theme_clean() + theme(legend.position = "none")

nurses <- nurses %>%
  mutate(state_lower = tolower(State))

Data Visualization


The graph below helps visual the entire country and see the differences in Average Salary across states. California leads with the highest, and we can see patches of dominance in the New England region, along with Washington, Oregon, and North Dakota.

states_map <- map_data("state")

nurses %>%
  ggplot(aes(fill=`Annual Salary Avg`)) +
  geom_map(aes(map_id=state_lower), color='gray', size=0.3, map=states_map) +
  expand_limits(x=states_map$long, y=states_map$lat) +
  theme_map() + 
  scale_fill_gradient(low = "azure", high = "blue")
